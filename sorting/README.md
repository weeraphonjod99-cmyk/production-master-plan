# Sorting Control

โปรแกรมเว็บแบบ offline สำหรับควบคุมงานรับเข้า เบิกออกไปคัด บันทึกผลคัด และดูยอดคงเหลืองาน HOLD ในรูปแบบ template ทั่วไป

## เปิดใช้งาน

เปิดไฟล์ `index.html` ด้วย browser ได้โดยตรง

## หน้าจอหลัก

- `ภาพรวม`: คิวงานคัด ยอด HOLD งานที่กำลังคัด และรายการเตือน
- `รับเข้างาน`: บันทึกงานเข้าพื้นที่ HOLD
- `เบิกไปคัด`: เลือกรายการรับเข้าและเบิกจำนวนไปยังพื้นที่คัด
- `ผลการคัด`: บันทึกงานดี งานเสีย และงานรอตัดสินใจ
- `ยอดคงเหลือ`: คำนวณรับเข้า เบิก HOLD คงเหลือ งานกำลังคัด และสถานะ
- `ข้อมูล`: ส่งออก/นำเข้า JSON และส่งออก CSV

## การเก็บข้อมูล

ข้อมูลถูกเก็บใน browser storage ของเครื่องที่เปิดใช้งาน ควรส่งออกไฟล์ JSON เป็นระยะเพื่อสำรองหรือย้ายเครื่อง
